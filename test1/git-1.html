git新建仓库并将本地仓库做的修改push到远程仓库

  • 1.首先在github上先新建一个仓库,比如test,将会生成一个地址，例如我的是https://github.com/figbir/test.git
  • 2.在本地新建一个文件夹test
  • 3.echo "# test" >> README.md （这是在本地新建并编辑README.md，其中“#test”就是README.md中的内容）
  • 4.git add README.md
  • 5.git commit -m "添加README.md"
  • 6.git remote add origin https://github.com/figbir/test.git
  • 7.git push origin master
